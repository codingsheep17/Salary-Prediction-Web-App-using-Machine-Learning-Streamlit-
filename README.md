# 💰 Salary Prediction Web App

A simple yet powerful Machine Learning web app that predicts salary based on user inputs.

Built using **Python, Scikit-learn, and Streamlit**, this project demonstrates end-to-end ML workflow — from data preprocessing to model deployment.

## 📌 Features

- 🔮 Real-time salary prediction
- 📊 Trained ML model (Regression)
- 🎯 Input-based predictions:
  - Age
  - Experience
  - Hours per week
  - Education level
- ⚡ Fast and interactive UI using Streamlit

---

## 🧠 How It Works

1. User enters input values
2. Data is preprocessed (including encoding)
3. Model predicts salary based on trained patterns
4. Result is displayed instantly
